# Vindi Ruby

Ruby toolkit para a [API de Recorrência][link-introducao-api] da [Vindi][link-vindi].

## Instalação

```ruby
gem 'vindi-ruby'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install vindi-ruby
